# Contributing To NotchSpark ✨

Thanks for wanting to make NotchSpark better.

## Great Contributions

- Short, surprising, accurate facts
- Smoother animation ideas
- Accessibility and readability improvements
- macOS packaging improvements
- Translations
- Bug reports with your macOS version and Mac model

## Development

```sh
swift test
swift build
Scripts/build_app.sh
open .build/NotchSpark.app
```

Please keep the app private, lightweight, and permission-friendly.

## Pull Requests

Before opening a PR:

- Run `swift test`
- Run `swift build`
- Keep UI changes compact and readable
- Avoid network features unless they are optional and privacy-preserving

Small, focused PRs are easiest to review.
